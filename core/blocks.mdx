Shipit comes with a few UI blocks that you can use in your project to speed up your development process.
You can find all the blocks in the `src/components/blocks` folder and it includes:
* FAQ (Frequently Asked Questions) Components
* Pricing Components
* Testimonials Components
* Features Components
* Footer Components
* Hero Components
* And more
